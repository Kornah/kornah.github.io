How to Install Discord on ANY Linux Distro

Introduction

This guide walks you through installing Discord on most Linux distros — whether you're using Ubuntu, Void Linux, Arch, or others. We'll cover both manual and automated methods with pros and cons.

Installation Methods

There are two common ways:

  • Official .tar.gz – Lightweight and distro-independent, but needs manual setup.
  • Flatpak – Easy to install via Flathub, but heavier and sandboxed.

Installing Discord via .tar.gz

1. Go to the official website and download the Linux version: discord.com/download

2. Extract the archive:

tar -xf discord-*.tar.gz

3. Move it to a system directory (optional):

sudo mv Discord /opt/discord

4. Launch with:

/opt/discord/Discord

5. (Optional) Create a desktop shortcut by making a .desktop file in ~/.local/share/applications/.

Installing Discord via Flatpak

1. Install Flatpak if you don’t have it already.

2. Run:

flatpak install flathub com.discordapp.Discord

3. To launch:

flatpak run com.discordapp.Discord

Flatpak provides sandboxing and easier updates but can use more memory.

Setting Up Auto-Start

Create a .desktop file in:

~/.config/autostart/discord.desktop

Example content:

[Desktop Entry]
Type=Application
Exec=/opt/discord/Discord
Hidden=false
NoDisplay=false
X-GNOME-Autostart-enabled=true
Name=Discord

Fixing Sound Issues

  • Check if Discord is sandboxed (Flatpak). Run with --no-sandbox if needed.
  • Make sure PulseAudio or PipeWire is working properly.
  • Try rebooting after initial setup.

FAQ

Why doesn't Discord start?

Run it in the terminal and check for missing libraries with ldd.

Can I use the web version instead?

Yes! Just go to discord.com/app in your browser.

Where’s the config directory?

It's usually in ~/.config/discord.

How to remove Discord completely?

Delete the installation directory and the config folder: 

rm -rf ~/.config/discord
sudo rm -rf /opt/discord
